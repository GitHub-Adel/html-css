ETIQUETA:
--------------------------------------
son las que encierran un contenido, hay dos tipos: etiqueta de apertura y de cierre.
 <etiqueta>Contenido</etiqueta>

hay etiquetas que no tienen contenido.
<etiqueta/>

las etiquetas de apertura <etiqueta> en ella se ponen los atributos y los eventos


CONTENIDO: 
--------------------------------------
es todo lo que se encuentra dentro de las etiquetas de apertura y cierre
<etiqueta>Contenido</etiqueta>


ATRIBUTO:
--------------------------------------
Se ponen en la etiqueta de apertura separados por espacio
<etiqueta atributo1="valor1 valor2" >  Contenido </etiqueta>
<input type="button" value="Sí">

si el atributo es de tipo boolean solo se pone el nombre.
<etiqueta visible> Contenido </etiqueta>
visible es de tipo boolean su funcion es hacer visible o no la etiqueta.


EVENTO:
--------------------------------------
los eventos se  utilizan para asociar una funcion() al elemento, se ponen 
en la etiqueta de apertura igual que los atributos, esta funcion() esta escrita
en javascript y es para programar lo que uno quiere que haga el elemento.
<etiqueta evento="funcion(parametro)"  >
<button onclick="CambiarColor(color)" >


ELEMENTO:
--------------------------------------
un elemento esta compuesto de todos los anterior 
<etiqueta atributo="valor" evento="valor" >Contenido</etiqueta>
todo esto es un elemento.
el elemento se llama como se llama la etiqueta.

hay elementos que solo tienen 1 etiqueta porque no tiene contenido
<etiqueta/>
